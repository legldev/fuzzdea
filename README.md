# fuzzdea Landing Page

Landing page preview for `fuzzdea.com`, built with React and Vite.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Stack

- React 19
- Vite 8
- ESLint 9

## Notes

The current page is a lightweight "coming soon" screen for the fuzzdea company website.
